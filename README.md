# Chip8Emulator

My first working emulator ever, made it by guides etc.
Wanna to continue and add more features

## Dependencies
```
g++
libsdl2
libsdl2-dev
make
```

## Compile
`make all`

## Usage
`emulator "Scale" "Speed" "ROM.ch8"`

## Example
`emulator 20 4 test_opcode.ch8`

Made in Visual Studio Code using make and g++
